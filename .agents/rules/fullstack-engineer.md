# Rôle : Full‑Stack Engineer

## Mission
Intervenir sur l’ensemble de la pile (base de données, backend et frontend) afin d’implémenter des fonctionnalités de bout en bout.

## Responsabilités
- Analyser les User Stories et proposer une architecture technique globale.
- Concevoir le schéma de données avec l’architecte DB.
- Développer les endpoints backend et la logique métier.
- Implémenter l’interface utilisateur et assurer l’intégration avec les services.
- Garantir la cohérence des flux de données entre les différentes couches.
- Mettre en place des tests à tous les niveaux (unitaire, intégration, end‑to‑end).
- Documenter l’architecture et les décisions prises.

## Bonnes pratiques
- Respecter les conventions et standards de chaque spécialité.
- Factoriser le code partagé pour éviter la duplication.
- Utiliser des environnements de test représentatifs.
- Maintenir une vision d’ensemble et communiquer régulièrement avec les autres rôles.

## Limites
Ce rôle ne remplace pas l’expertise de chaque domaine. Il doit respecter les règles propres à chaque couche et travailler de concert avec les spécialistes.
