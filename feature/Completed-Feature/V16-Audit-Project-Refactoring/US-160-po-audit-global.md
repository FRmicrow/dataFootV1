# US-160: Audit global et plan d'action détaillé

**Rôle**: Product Owner / Full-Stack Engineer

**Objectif**: En tant que Product Owner, je veux réaliser un audit exhaustif de l'application au regard des nouveaux standards afin de définir un plan de refactoring précis et priorisé.

**Tâches**:
- [ ] Analyser le dossier `/agent` (rules, skills, workflows) pour lister tous les standards applicables.
- [ ] Parcourir l'arborescence du projet (Frontend, Backend, DB, ML, DevOps).
- [ ] Identifier les fichiers et modules ne respectant pas les principes SOLID, Clean Code et d'esthétique premium.
- [ ] Documenter les points de friction, les dettes techniques et les opportunités d'amélioration.
- [ ] Produire un rapport d'audit détaillé avec des recommandations par domaine.

**Exigences**:
- Couvrir 100% des modules de l'application.
- Utiliser la grille de lecture des `rules` et `skills` définis dans le dossier `.agents`.
- Le plan d'action doit être découpé par spécialité (Front, Back, DB, ML, DevOps).

**Critères d'acceptation**:
- [ ] Un document de synthèse d'audit est disponible dans le dossier de la feature.
- [ ] Le plan d'action est validé et prêt à être décliné en tâches techniques.
- [ ] Chaque domaine d'intervention est clairement identifié avec ses enjeux.
