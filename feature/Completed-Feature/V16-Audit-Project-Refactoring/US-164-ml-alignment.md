# US-164: Optimisation des pipelines et modèles ML

**Rôle**: Machine Learning Engineer

**Objectif**: En tant que Machine Learning Engineer, je veux aligner les services de prédiction sur les nouveaux standards de performance et d'interprétabilité.

**Tâches**:
- [ ] Auditer les scripts d'entraînement et d'inférence.
- [ ] Améliorer l'ingénierie des caractéristiques (feature engineering).
- [ ] Optimiser les hyperparamètres et la validation croisée.
- [ ] Documenter les performances des modèles et l'importance des variables.
- [ ] Assurer le versioning des modèles et des données.

**Exigences**:
- Respecter les instructions de `machine-learning-engineer.md`.
- Favoriser des modèles performants sur des datasets tabulaires (type CatBoost).

**Critères d'acceptation**:
- [ ] Les services de prédiction sont stables et performants.
- [ ] La documentation technique du modèle est à jour.
- [ ] Le pipeline ML est reproductible.
