# US-175 : Intégration des couleurs sur la page Joueur

**En tant que** Frontend Engineer
**Je veux** que la page joueur reflète les couleurs de son club actuel
**Afin de** renforcer l'immersion et le lien entre le joueur et son équipe.

## Tâches
- [x] Modifier `PlayerProfilePageV3.jsx` pour extraire les couleurs de l'équipe courante du joueur.
- [x] Passer ces couleurs au composant `ProfileHeader`.

## Exigences
- Si le joueur est un agent libre, utiliser le thème neutre par défaut.

## Critères d'acceptation
- La page joueur affiche le dégradé de couleurs de son club actuel dans le header.
