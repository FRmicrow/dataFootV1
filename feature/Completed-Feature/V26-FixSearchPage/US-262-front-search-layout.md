# User Story: US-262 - Frontend Search Layout & Data Binding

**En tant que** Frontend Engineer,
**Je veux** restaurer l'affichage en deux colonnes et le formulaire de recherche,
**Afin de** retrouver l'expérience utilisateur initiale du "Scout Engine".

## Tâches
- [ ] Vérifier le binding des données entre la réponse API et les composants `Card` et `Grid`.
- [ ] S'assurer que les images se chargent correctement avec les nouveaux noms de champs.
- [ ] Restaurer la visibilité du composant `CountrySelector`.

## Critères d'Acceptation
- Le formulaire de recherche est visible et fonctionnel.
- Les résultats s'affichent en deux colonnes : Clubs à gauche, Players à droite.
- Les drapeaux s'affichent à côté du pays/de la nationalité.
