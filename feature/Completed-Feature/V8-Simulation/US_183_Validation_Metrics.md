# US_183: Validation Metrics (Quant)

## Mission
Compute statistical and financial metrics to evaluate model and strategy performance.

## Mission Details
1. **Statistical Metrics**:
    - **Log-loss**: Accuracy of probability predictions.
    - **Brier Score**: Quadratic error of predictions.
    - **Calibration Curve**: Reliability of probabilities.
2. **Financial Metrics**:
    - **ROI**: Return on Investment per strategy.
    - **Profit Curve**: Cumulative PnL over time.
    - **Sharpe Ratio**: Risk-adjusted returns.

## Success Criteria
- [ ] Automated report generation for a simulation run.
- [ ] Exportable metrics for model comparison.
