# 👤 US_184: Simulation Dashboard UI
**Accountable Agent**: Frontend Developer
**Feature Type**: High-Density Dashboard
**Mission**: Create a professional analytical interface for browsing and launching backtesting Forge runs.

---

## 🎯 Strategic Objective
Provide a "Control Center" where the user can manage their model evolution roadmap and compare different league performances.

## 📋 Functional Requirements
- **Sim-Browser**: Select League, Season, and Model Version.
- **Top-Line Metrics**: Global Accuracy, Brier, and ROI badges.
- **Accuracy Heatmap**: Visual grid showing model performance by matchday.
- **ROI Evolution Chart**: Cumulative growth curve of the theoretical bankroll.

## ✅ Acceptance Criteria
- User can launch a Forge run from the dashboard.
- Dashboard shows "Last Completed Month" and "Current Accuracy Trend".
- Responsive design for high-resolution screens (institutional focus).
